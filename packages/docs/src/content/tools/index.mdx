# Tools Overview

Pacifica MCP provides **22 tools** grouped into four categories. Market data and account tools are free and cached (5-minute TTL). Trading tools require a wallet and sign every request with your Ed25519 keypair.

## Market Data

Free — no authentication required. Results cached for 5 minutes.

| Tool | Description | Auth |
|---|---|---|
| `pacifica-markets` | List all markets or look up a single market by symbol | None |
| `pacifica-prices` | Real-time prices, funding rate, open interest, 24h volume | None |
| `pacifica-orderbook` | Order book bids and asks with configurable aggregation | None |
| `pacifica-candles` | OHLCV candlestick data, 1m to 1d intervals | None |
| `pacifica-recent-trades` | Most recent trades for a market | None |
| `pacifica-funding-rates` | Historical funding rate entries | None |

## Account Monitoring

Free — no authentication required. Uses the MCP wallet address by default, or any Solana address you provide.

| Tool | Description | Auth |
|---|---|---|
| `pacifica-account` | Balance, equity, margin used, fee tier | None |
| `pacifica-positions` | Open positions with entry price, margin, and PnL | None |
| `pacifica-orders` | All open orders | None |
| `pacifica-order-history` | Historical orders with status | None |
| `pacifica-trade-history` | Historical trades with realized PnL and fees | None |
| `pacifica-portfolio` | Portfolio equity over time (1d / 7d / 30d / all) | None |
| `pacifica-balance-history` | Balance events: deposits, withdrawals, funding payments | None |

## Trading

Requires a wallet. Every request is signed with your Ed25519 keypair before being sent to Pacifica.

| Tool | Description | Auth |
|---|---|---|
| `pacifica-market-order` | Place a market order (bid = long, ask = short) | Wallet |
| `pacifica-limit-order` | Place a limit order with GTC / IOC / ALO / TOB | Wallet |
| `pacifica-stop-order` | Place a stop-market or stop-limit order | Wallet |
| `pacifica-set-tpsl` | Set take-profit and/or stop-loss on a position | Wallet |
| `pacifica-cancel-order` | Cancel a specific order or all open orders for a symbol | Wallet |
| `pacifica-set-leverage` | Set leverage multiplier for a market (1–100x) | Wallet |
| `pacifica-set-margin-mode` | Switch between cross margin and isolated margin | Wallet |

## System

| Tool | Description | Auth |
|---|---|---|
| `pacifica-wallet` | Show wallet address, network, and balance | None |
| `pacifica-tools` | List all available tools grouped by category | None |
