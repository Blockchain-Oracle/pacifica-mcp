# Account Tools

All account tools are free and require no authentication. They use the MCP wallet address by default — pass any Solana address to query other accounts. Results are cached for 5 minutes.

---

## pacifica-account

Get account overview: balance, equity, margin used, available margin, fee level.

**Parameters**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `account` | string | No | Solana wallet address. Defaults to MCP wallet. |

**Example prompt**

> *"What's my account balance and available margin?"*

**Example response**

```json
{
  "account": "7xK...abc",
  "balance": "1000.00",
  "equity": "1023.45",
  "margin_used": "67.42",
  "available_margin": "955.58",
  "positions_count": 1,
  "fee_level": "taker"
}
```

## pacifica-positions

Get all open positions for an account.

**Parameters**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `account` | string | No | Solana wallet address. Defaults to MCP wallet. |

**Example prompt**

> *"Show me my open positions."*

> *"What positions does 7xK...abc have open?"*

**Example response**

```json
[
  {
    "symbol": "BTC-PERP",
    "side": "bid",
    "amount": "0.01",
    "entry_price": "67421.00",
    "mark_price": "67450.00",
    "unrealized_pnl": "0.29",
    "margin": "67.42",
    "funding": "-0.05",
    "leverage": "10"
  }
]
```

## pacifica-order-history

Get historical orders for an account.

**Parameters**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `account` | string | No | Solana wallet address. Defaults to MCP wallet. |
| `limit` | integer | No | Number of orders, 1–500 (default `50`) |

**Example prompt**

> *"Show me my last 20 orders."*

---

## pacifica-trade-history

Get historical trades with realized PnL and fees.

**Parameters**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `account` | string | No | Solana wallet address. Defaults to MCP wallet. |
| `symbol` | string | No | Filter by market (e.g. `BTC-PERP`) |
| `limit` | integer | No | Number of trades, 1–500 (default `50`) |

**Example prompt**

> *"Show me my BTC-PERP trade history with PnL."*

**Example response**

```json
[
  {
    "trade_id": "trd_xyz789",
    "symbol": "BTC-PERP",
    "side": "ask",
    "amount": "0.01",
    "price": "67800.00",
    "realized_pnl": "3.79",
    "fee": "0.034",
    "timestamp": "1712699500000"
  }
]
```

## pacifica-portfolio

Get portfolio equity history over time.

**Parameters**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `account` | string | No | Solana wallet address. Defaults to MCP wallet. |
| `time_range` | enum | No | `1d`, `7d`, `14d`, `30d`, `all` (default `7d`) |

**Example prompt**

> *"Show me my portfolio performance over the last 30 days."*

**Example response**

```json
[
  { "timestamp": "1712613600000", "equity": "987.00" },
  { "timestamp": "1712700000000", "equity": "1023.45" }
]
```

## pacifica-balance-history

Get balance event history: deposits, withdrawals, trades, and funding payments.

**Parameters**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `account` | string | No | Solana wallet address. Defaults to MCP wallet. |
| `limit` | integer | No | Number of events, 1–500 (default `50`) |

**Example prompt**

> *"Show me my recent deposits and withdrawals."*

**Example response**

```json
[
  {
    "type": "deposit",
    "amount": "500.00",
    "timestamp": "1712600000000"
  },
  {
    "type": "funding",
    "amount": "-0.05",
    "symbol": "BTC-PERP",
    "timestamp": "1712692800000"
  }
]
```
