# Market Data Tools

All market data tools are free and require no authentication. Results are cached for 5 minutes.

---

## pacifica-markets

List all available perpetual markets, or look up a single market by symbol.

**Parameters**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `symbol` | string | No | Filter to a single market (e.g. `BTC-PERP`) |

**Example prompt**

> *"List all available markets on Pacifica."*

> *"Show me the BTC-PERP market details — tick size, lot size, max leverage."*

**Example response**

```json
{
  "symbol": "BTC-PERP",
  "tick_size": "0.5",
  "lot_size": "0.001",
  "max_leverage": "100",
  "min_order_size": "0.001",
  "maker_fee": "0.0002",
  "taker_fee": "0.0005"
}
```

## pacifica-prices

Get real-time prices for all markets, or filter to a single symbol.

**Parameters**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `symbol` | string | No | Filter to a single symbol (e.g. `ETH-PERP`) |

**Example prompt**

> *"What's the current BTC-PERP price?"*

> *"Show me prices and funding rates for all markets."*

**Example response**

```json
{
  "symbol": "BTC-PERP",
  "mark": "67420.50",
  "oracle": "67415.00",
  "mid": "67418.00",
  "funding_rate": "0.000023",
  "next_funding": "1712700000000",
  "open_interest": "89000000",
  "volume_24h": "142000000"
}
```

## pacifica-orderbook

Get the order book (bids and asks) for a market with configurable price aggregation.

**Parameters**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `symbol` | string | Yes | Market symbol (e.g. `BTC-PERP`) |
| `agg_level` | enum | No | Price bucket aggregation: `1`, `10`, `100`, `1000`, `10000` (default `1`) |

**Example prompt**

> *"Show me the BTC-PERP order book."*

> *"What does the ETH-PERP order book look like at $10 aggregation?"*

**Example response**

```json
{
  "bids": [
    ["67419.0", "1.234"],
    ["67418.5", "0.567"]
  ],
  "asks": [
    ["67421.0", "0.890"],
    ["67421.5", "2.100"]
  ]
}
```

## pacifica-candles

Get OHLCV candlestick data for a market.

**Parameters**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `symbol` | string | Yes | Market symbol (e.g. `SOL-PERP`) |
| `interval` | enum | Yes | `1m`, `3m`, `5m`, `15m`, `30m`, `1h`, `2h`, `4h`, `8h`, `12h`, `1d` |
| `limit` | integer | No | Number of candles, 1–1000 (default `100`) |

**Example prompt**

> *"Show me the last 50 hourly candles for BTC-PERP."*

> *"Get the 4h OHLCV data for ETH-PERP."*

**Example response**

```json
[
  {
    "time": "1712696400000",
    "open": "67200.00",
    "high": "67500.00",
    "low": "67100.00",
    "close": "67420.00",
    "volume": "145.23"
  }
]
```

## pacifica-funding-rates

Get historical funding rates for a market.

**Parameters**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `symbol` | string | Yes | Market symbol (e.g. `BTC-PERP`) |
| `limit` | integer | No | Number of entries, 1–500 (default `20`) |

**Example prompt**

> *"Show me the last 10 funding rates for BTC-PERP."*

> *"What has the ETH-PERP funding rate been doing over the past week?"*

**Example response**

```json
[
  {
    "symbol": "BTC-PERP",
    "funding_rate": "0.000023",
    "timestamp": "1712692800000"
  },
  {
    "symbol": "BTC-PERP",
    "funding_rate": "0.000018",
    "timestamp": "1712664000000"
  }
]
```
