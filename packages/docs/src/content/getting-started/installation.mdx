# Installation

Pacifica MCP runs as a local process via `npx`. No global install needed — just add it to your MCP client.

**npm packages:** [`@pacifica-dev/mcp`](https://www.npmjs.com/package/@pacifica-dev/mcp) · [`@pacifica-dev/cli`](https://www.npmjs.com/package/@pacifica-dev/cli)

## Claude Code

The fastest way. Run this in your terminal:

```bash
claude mcp add pacifica npx @pacifica-dev/mcp
```

That's it. Restart Claude Code and start chatting. Try: *"Show me the BTC-PERP price."*

### Verify it's working

```bash
claude mcp list
```

You should see `pacifica-mcp` in the list.

## Claude Desktop

Open your Claude Desktop config file:

- **macOS:** `~/Library/Application Support/Claude/claude_desktop_config.json`
- **Windows:** `%APPDATA%\Claude\claude_desktop_config.json`

Add the Pacifica MCP entry:

```json
{
  "mcpServers": {
    "pacifica-mcp": {
      "command": "npx",
      "args": ["-y", "@pacifica-dev/mcp"]
    }
  }
}
```

Save the file and restart Claude Desktop.

---

## Cursor

Open **Cursor Settings** → **MCP** → **Add MCP Server**.

Or edit `~/.cursor/mcp.json` directly:

```json
{
  "mcpServers": {
    "pacifica-mcp": {
      "command": "npx",
      "args": ["-y", "@pacifica-dev/mcp"]
    }
  }
}
```

Restart Cursor after saving.

## Environment variables

| Variable | Default | Description |
|---|---|---|
| `PACIFICA_NETWORK` | `testnet` | `testnet` or `mainnet` |
| `PACIFICA_PRIVATE_KEY` | (auto-generated) | Base58-encoded Ed25519 private key override |

To trade on mainnet, set `PACIFICA_NETWORK=mainnet` in the MCP server environment.

### Claude Desktop with env vars

```json
{
  "mcpServers": {
    "pacifica-mcp": {
      "command": "npx",
      "args": ["-y", "@pacifica-dev/mcp"],
      "env": {
        "PACIFICA_NETWORK": "mainnet"
      }
    }
  }
}
```

## Standalone CLI

For terminal use without an MCP host:

```bash
npm install -g @pacifica-dev/cli
pacifica prices --symbol BTC-PERP
```

See the [CLI guide](/guides/cli) for full usage.

## Wallet setup

The first time the MCP server starts, it auto-generates an Ed25519 wallet at `~/.pacifica-mcp/config.json`.

**Backup this file.** It contains your private key.

To check your wallet address:

> *"Show me my Pacifica wallet address."*

Then deposit USDC to that address on Pacifica to start trading.

For testnet setup, see the [Testnet guide](/guides/testnet).
