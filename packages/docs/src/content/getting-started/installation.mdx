# Installation

Pacifica MCP installs in two ways: as an **MCP server** (every major agent client supports it) or as a **standalone CLI** (for terminal use). Pick whichever fits your workflow — or install both.

## Install the MCP server

Pacifica MCP works natively with 12+ MCP clients. Pick yours:

<Tabs items={['Claude Code', 'Claude Desktop', 'Cursor', 'Windsurf', 'VS Code', 'Gemini CLI', 'Codex', 'Zed', 'Continue', 'Cline', 'Goose', 'OpenClaw']}>

<Tabs.Tab>
```bash
claude mcp add pacifica npx @pacifica-dev/mcp
```

Claude Code downloads and runs the server automatically on next session.
</Tabs.Tab>

<Tabs.Tab>
Open your config file:
- macOS: `~/Library/Application Support/Claude/claude_desktop_config.json`
- Windows: `%APPDATA%\Claude\claude_desktop_config.json`
- Linux: `~/.config/Claude/claude_desktop_config.json`

```json
{
  "mcpServers": {
    "pacifica": {
      "command": "npx",
      "args": ["-y", "@pacifica-dev/mcp"]
    }
  }
}
```

Fully quit and restart Claude Desktop.
</Tabs.Tab>

```json
  }
```

Or use the UI: Settings → Tools & MCP → Add new MCP server.
</Tabs.Tab>

## Environment variables

| Variable | Default | Description |
|---|---|---|
| `PACIFICA_NETWORK` | `testnet` | `testnet` or `mainnet` |
| `PACIFICA_PRIVATE_KEY` | (auto-generated) | Base58-encoded Ed25519 private key override |
| `LOG_LEVEL` | `info` | `debug`, `info`, `warn`, `error` |

To set env vars, add an `env` block to your MCP config:

```json
{
  "mcpServers": {
    "pacifica": {
      "command": "npx",
      "args": ["-y", "@pacifica-dev/mcp"],
      "env": {
        "PACIFICA_NETWORK": "mainnet"
      }
    }
  }
}
```

For terminal use without an MCP host:


## Wallet setup

The first time the MCP server starts, it auto-generates an Ed25519 wallet at `~/.pacifica-mcp/config.json`.

**Backup this file.** It contains your private key.

To deposit funds and start trading:

1. Copy your private key from `~/.pacifica-mcp/config.json`
2. Import it into a browser wallet ([Phantom](https://phantom.app) or [Backpack](https://backpack.app))
3. Go to [test-app.pacifica.fi](https://test-app.pacifica.fi) (access code: `Pacifica`) and connect your wallet
4. Deposit SOL, USDC, or any supported asset

For the full walkthrough, see the [Testnet guide](/guides/testnet).
