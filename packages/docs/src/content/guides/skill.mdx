# Agent Skill

Pacifica also ships as an **Agent Skill** -- a `SKILL.md` file that teaches AI agents when and how to use Pacifica tools, without consuming context until needed.

## Skill vs MCP server -- what is the difference?

They are complementary. Install both for the best experience.

| | MCP Server | Skill |
| --- | --- | --- |
| What it ships | Tool schemas (machine-readable) | Procedural instructions (markdown) |
| Context cost | Always in context | Minimal at startup, full body loads on demand |
| Who invokes | Agent calls tools directly | Agent reads instructions, then calls tools |
| Install | `claude mcp add pacifica npx @pacifica-dev/mcp` | `~/.claude/skills/pacifica/SKILL.md` |

The skill teaches your agent:
- When Pacifica tools are the right choice
- The decision tree mapping user intent to tool
- Parameter conventions (symbol format, side values, amount as string)
- Free vs wallet-required tools
- Example trading workflows

## Three install methods

### 1. Via the community skills CLI

Uses [`vercel-labs/skills`](https://github.com/vercel-labs/skills), the emerging standard for skill distribution:

```bash
pnpm dlx skills add github:Blockchain-Oracle/pacifica-mcp --skill pacifica

# or with npm / bun
npx skills add github:Blockchain-Oracle/pacifica-mcp --skill pacifica
bunx skills add github:Blockchain-Oracle/pacifica-mcp --skill pacifica
```

This clones the skills directory and installs `SKILL.md` into the right location for whichever agents you have installed.

### 2. Via prompt

Paste this sentence into any agent:

```
Read https://pacifica-mcp.xyz/skill.md and follow the instructions to install the Pacifica MCP skill.
```

The agent fetches the file over HTTPS, reads the instructions, and installs the skill itself. No CLI required.

### 3. Manual download

Download `SKILL.md` from the [skill page](https://pacifica-mcp.xyz/skill) and place it in the correct path for your agent:

| Client | Path |
| --- | --- |
| Claude Code | `~/.claude/skills/pacifica/SKILL.md` |
| Cursor (2.4+) | `~/.cursor/skills/pacifica/SKILL.md` |
| Windsurf | `~/.codeium/windsurf/skills/pacifica/SKILL.md` |
| VS Code Copilot | `~/.copilot/skills/pacifica/SKILL.md` |
| Gemini CLI | `~/.gemini/skills/pacifica/SKILL.md` |
| Cline (3.48+) | `~/.cline/skills/pacifica/SKILL.md` |
| Goose | `~/.config/agents/skills/pacifica/SKILL.md` |

## What the skill contains

The SKILL.md frontmatter tells the agent when to activate:

```yaml
---
name: pacifica
description: Live market data and trading actions for Pacifica...
---
```

The body includes:
- **When to use / when not to use** -- trigger conditions and skip conditions
- **Decision tree** -- maps 31 user intents to specific tools
- **Parameter guide** -- symbol format, side values, amount conventions
- **Free vs wallet-required** -- which tools need signing
- **Example workflows** -- multi-step trading scenarios
- **Error handling** -- common errors and how to resolve them
