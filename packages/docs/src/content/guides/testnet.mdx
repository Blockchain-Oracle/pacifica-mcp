# Testnet Guide

Pacifica testnet lets you trade with testnet funds at no risk. It's the best way to try the MCP server before moving real funds.

## Access

Pacifica testnet requires an access code:

**Access code: `Pacifica`**

Use this code when registering at [test-app.pacifica.fi](https://test-app.pacifica.fi).

---

## Step 1: Configure the MCP server for testnet

Testnet is the default. No extra configuration needed.

To explicitly confirm:

```json
{
  "mcpServers": {
    "pacifica-mcp": {
      "command": "npx",
      "args": ["-y", "@pacifica-dev/mcp"],
      "env": {
        "PACIFICA_NETWORK": "testnet"
      }
    }
  }
}
```

---

## Step 2: Get your wallet address

Ask Claude:

> *"What's my Pacifica wallet address?"*

The MCP server returns your Ed25519 public key. Copy it — you'll need it in the next step.

---

## Step 3: Fund your testnet account

Deposits happen through the Pacifica web app — not through the MCP or CLI.

### 3a. Get your private key

Your private key is stored at `~/.pacifica-mcp/config.json`. Open the file and copy the `privateKey` value:

```bash
cat ~/.pacifica-mcp/config.json
```

### 3b. Import into a browser wallet

1. Install [Phantom](https://phantom.app) or [Backpack](https://backpack.app) browser extension
2. Switch the wallet to **Solana devnet/testnet**
3. Choose **Import Private Key** (not seed phrase)
4. Paste the `privateKey` value from the config file

### 3c. Connect to Pacifica testnet

1. Go to [test-app.pacifica.fi](https://test-app.pacifica.fi)
2. Enter the access code: **`Pacifica`**
3. Connect your browser wallet (Phantom/Backpack)

### 3d. Deposit

1. Use the faucet on the Pacifica testnet app to get testnet SOL
2. Deposit SOL, USDC, or any supported asset through the Pacifica deposit interface

Testnet funds have no real value. Claim and deposit as much as you need.

---

## Step 4: Verify your balance

After depositing:

> *"Show me my Pacifica account balance."*

You should see your balance reflected in the account overview.

---

## Step 5: Start trading

All market data on testnet mirrors mainnet prices in real time. Trading mechanics are identical.

**Try it:**

> *"What's the BTC price on testnet?"*

> *"Buy 0.01 BTC at market price."*

> *"Show me my open positions."*

---

## Testnet API

The MCP server automatically uses the correct API endpoint:

| Network | API endpoint |
|---|---|
| Testnet | `https://test-api.pacifica.fi/api/v1` |
| Mainnet | `https://api.pacifica.fi/api/v1` |

---

## Switching to mainnet

When you're ready to trade with real funds:

1. Deposit funds (e.g. USDC, SOL) to your wallet address on mainnet Pacifica
2. Update your MCP config: set `PACIFICA_NETWORK=mainnet`
3. Restart your MCP client

The same wallet file works on both networks.

> **Warning:** Mainnet trades use real money. Double-check all order parameters before confirming.
