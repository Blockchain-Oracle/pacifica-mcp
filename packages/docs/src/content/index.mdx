# Pacifica MCP

**Pacifica MCP** is a [Model Context Protocol](https://modelcontextprotocol.io) server that gives AI agents the ability to trade, monitor, and manage positions on [Pacifica Finance](https://pacifica.fi) — the #1 perpetuals DEX on Solana.

Connect it to Claude Code, Cursor, Windsurf, VS Code Copilot, OpenClaw, Gemini CLI, or any MCP-compatible client and start trading with natural language.

## What can it do?

Pacifica MCP exposes **36 tools** across five categories:

| Category | Tools | Auth |
|---|---|---|
| Market Data | Prices, order book, candles, funding rates, recent trades | Free |
| Account Monitoring | Positions, orders, trade history, portfolio equity | Free |
| Trading | Market/limit/stop orders, TP/SL, batch, edit, cancel, leverage, margin mode | Wallet |
| Account Management | Withdraw, subaccounts, fund transfers | Wallet |
| Real-Time | WebSocket streaming for prices, trades, orderbook, account updates | Free |
| System | Wallet info, list tools | Free |

### Example prompts

- *"What's the current BTC price and funding rate?"*
- *"Show me my open positions and unrealized PnL."*
- *"Open a 0.01 BTC long at market with 10x leverage."*
- *"Set a stop-loss on my ETH position at $2,100."*
- *"Watch BTC trades for 10 seconds and summarize."*

## Quick install

**MCP server** — one command for any supported client:

```bash
# Claude Code
claude mcp add pacifica npx @pacifica-dev/mcp

# Gemini CLI
gemini mcp add pacifica npx -y @pacifica-dev/mcp

# OpenClaw
openclaw mcp set pacifica '{"command":"npx","args":["-y","@pacifica-dev/mcp"]}'
```

For Cursor, Windsurf, VS Code, Zed, Continue, Cline, and Goose — see the [Installation guide](/getting-started/installation).

**Agent Skill** — teaches your agent when and how to use each tool:

```bash
pnpm dlx skills add github:Blockchain-Oracle/pacifica-mcp --skill pacifica
```

**Standalone CLI** — terminal-native, no AI required:

```bash
npm install -g @pacifica-dev/cli
```

## npm Packages

- [`@pacifica-dev/cli`](https://www.npmjs.com/package/@pacifica-dev/cli) — CLI + core library (36 tools)
