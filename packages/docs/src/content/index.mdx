# Pacifica MCP

**Pacifica MCP** is a [Model Context Protocol](https://modelcontextprotocol.io) server that gives AI agents the ability to trade, monitor, and manage positions on [Pacifica Finance](https://pacifica.fi) — the #1 perpetuals DEX on Solana.

Connect it to Claude, Cursor, or any MCP-compatible client and start trading with natural language.

## What can it do?

Pacifica MCP exposes **36 tools** across four categories:

| Category | Tools | Auth |
|---|---|---|
| Market Data | Prices, order book, candles, funding rates, recent trades | None |
| Account Monitoring | Positions, orders, trade history, portfolio equity | None |
| Trading | Market/limit/stop orders, TP/SL, batch, edit, cancel, leverage, margin mode | Wallet required |
| Account Management | Withdraw, subaccounts, fund transfers | Wallet required |
| System | Wallet info, list tools | None |

### Example prompts

- *"What's the current BTC-PERP price and funding rate?"*
- *"Show me my open positions and unrealized PnL."*
- *"Buy 0.01 BTC-PERP at market price."*
- *"Set a stop-loss on my ETH-PERP long at $2800."*
- *"Cancel all my open orders on SOL-PERP."*

## Quick install

**MCP server** — add to Claude Code:

```bash
claude mcp add pacifica npx @pacifica-dev/mcp
```

**Agent Skill** — cross-client install via the community skills CLI:

```bash
pnpm dlx skills add github:Blockchain-Oracle/pacifica-mcp --skill pacifica
```

**Standalone CLI** — install globally:

```bash
npm install -g @pacifica-dev/cli
```

## npm Packages

- [`@pacifica-dev/mcp`](https://www.npmjs.com/package/@pacifica-dev/mcp) — MCP server (thin wrapper)
- [`@pacifica-dev/cli`](https://www.npmjs.com/package/@pacifica-dev/cli) — CLI + core library (36 tools)
